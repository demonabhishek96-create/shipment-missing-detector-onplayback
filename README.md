# 🚀 Shipment Missing Detector (AI CCTV Forensics Tool)

An AI-powered tool to detect **when a shipment goes missing** from CCTV footage with exact timestamp and proof.

---

## 🔍 Problem

In warehouse operations, identifying when a shipment (box/trolley/pallet) goes missing is:
- Manual
- Time-consuming (hours of video scanning)
- Error-prone

---

## ✅ Solution

This tool automatically:
- Detects shipment movement/disappearance
- Provides exact timestamp
- Generates proof frames

---

## 🎯 Key Features

- 📍 ROI-based detection (focus on specific area)
- ⏱ Exact timestamp identification
- 🖼 Proof frames (before & after)
- 🚫 Ignores human movement
- ⚡ Fast scanning (multi-stage detection)

## ⚙️ How It Works

1. Load CCTV video through RTSP
2. Select ROI (Region of Interest)
3. Run detection
4. Get:
   - Movement timestamp
   - Frames
